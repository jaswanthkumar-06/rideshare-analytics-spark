# RideShare Analytics Data Engineering Project

## Overview
This project simulates a ride-sharing analytics data warehouse using PySpark and PostgreSQL.

## Architecture
Raw CSV → PySpark Cleaning → Warehouse Tables → SQL Analytics → Pandas Reporting

## Tech Stack
Python | PySpark | PostgreSQL | SQL | Pandas | Matplotlib

## Features
- Data cleaning using PySpark
- Feature engineering
- Star schema design
- Advanced SQL analytics
- Revenue reporting

## How to Run
1. Install PySpark
2. Run clean_transform.py
3. Create tables using schema.sql
4. Insert data into fact_rides
5. Run analysis queries
6. Execute revenue_report.py
